## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to uncover insights about:
- Distribution of Movies vs TV Shows
- Content ratings
- Movie duration patterns
- Release trends over the years
- Top contributing countries

## Dataset
- Netflix Movies and TV Shows Dataset (`netflix_titles.csv`)

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Movies dominate Netflix content compared to TV Shows
- Most content is released after 2015
- The US contributes the highest number of titles
